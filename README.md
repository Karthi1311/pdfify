# pdfify

A lightweight Node.js + Express microservice for converting uploaded files to PDF. Includes a small web UI in `public/` and server logic in `server.js`.

## Features

- Upload files via a simple web UI
- Convert uploaded files to PDF
- Download converted PDFs
- Minimal, easy-to-run codebase

## Quickstart

1. Install dependencies:

   npm install

2. Run the server:

   node server.js

3. Open http://localhost:3000 (or the port configured in `server.js`) and upload a file.
